FEM Coding Assignments
======================

This repository contains the coding assignments for EM525 at Iowa State University.
Each assignment is stored in its own branch:

* [**Assignment 1**](https://github.com/solidsgroup/EM525/tree/ps1): Introduction to linux and C++
* [**Assignment 2**](https://github.com/solidsgroup/EM525/tree/ps2): CST shape functions
* [**Assignment 3**](https://github.com/solidsgroup/EM525/tree/ps3): LST, Q4, Q9 shape functions
* [**Assignment 4**](https://github.com/solidsgroup/EM525/tree/ps4): Quadrature Points and the isoparametric element Jacobian
* [**Assignment 5**](https://github.com/solidsgroup/EM525/tree/ps5): Elastic strain energy density,and element energy, and derivatives
* [**Assignment 6**](https://github.com/solidsgroup/EM525/tree/ps6): Isoparametric elements and unstructured meshes
* [**Assignment 7**](https://github.com/solidsgroup/EM525/tree/ps7): Full FEM simulation
* [**Assignment 8**](https://github.com/solidsgroup/EM525/tree/ps8): Stresses and Convergence Study
